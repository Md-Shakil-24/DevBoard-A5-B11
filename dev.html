Question-1: What are the different ways to select an element in the DOM?

----------------------------------------------------------------------------------------------------------------------------------------

You can select elements using methods like getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll. These methods allow selection by ID, class, tag, or more complex CSS-style selectors.

Question-2: What is the difference between innerHTML, innerText, and textContent?

----------------------------------------------------------------------------------------------------------------------------------------

innerHTML gets or sets HTML content, including tags. innerText retrieves visible text, ignoring HTML and styles. textContent gets or sets the text content, including hidden elements, without considering HTML or styles.

Question-3: What is event delegation in the DOM?

----------------------------------------------------------------------------------------------------------------------------------------

Event delegation is a technique where you attach a single event listener to a parent element, which handles events for child elements. This improves performance and simplifies code, especially when dynamically adding or removing elements.

Question-4: What is event bubbling in the DOM?

----------------------------------------------------------------------------------------------------------------------------------------

Event bubbling is the process where an event triggered on a child element propagates upward through its parent elements in the DOM. This allows multiple elements to handle the same event, starting from the target to the root.

Question-5: How do you create, add, and remove elements using JavaScript?

----------------------------------------------------------------------------------------------------------------------------------------

To create an element, use document.createElement(). Add it with appendChild() or insertBefore(). To remove, use removeChild() or parentElement.removeChild(). You can also use remove() directly on the element..
